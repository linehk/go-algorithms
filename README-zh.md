[English](./README.md "English") | 简体中文

go-algorithms 是一个保存了 Go 语言实现的数据结构和算法的仓库。

这个仓库未遵循 Go 语言的最佳实践规范，只是为了学习数据结构和算法。

## 安装

```bash
git clone https://github.com/linehk/go-algorithms.git
```
